# Tessdata Files

This folder contains the Tesseract OCR trained data files.

## Pre-built Releases

**`eng.traineddata` is already included** - no additional download needed.

## Building from Source

If building from source, download `eng.traineddata` and place it here:

1. **Best Quality (Recommended)**: [tessdata_best](https://github.com/tesseract-ocr/tessdata_best)
   - Direct link: https://github.com/tesseract-ocr/tessdata_best/raw/main/eng.traineddata
   - File size: ~15 MB

2. **Fast Mode** (alternative): [tessdata_fast](https://github.com/tesseract-ocr/tessdata_fast)
   - Direct link: https://github.com/tesseract-ocr/tessdata_fast/raw/main/eng.traineddata
   - File size: ~4 MB

The `best` version provides higher accuracy (recommended for game text).
